# CRUD API Endpoints

| Endpoint                                | Type   | Operation                    |
|-----------------------------------------|--------|------------------------------|
| `/menu/:id`                             | GET    | Get menu for a restaurant    |
| `/menu`                                 | POST   | Add menu for a restaurant    |
| `/menu/:id`                             | PUT    | Update menu for a restaurant |
| `/menu/:id`                             | DELETE | Delete menu for a restaurant |
